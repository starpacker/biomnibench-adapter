### 失败案例 1: 32864625_1 - 数据类型不匹配

**错误日志**：
```
KeyError: 250
```

**AI 代码**：
```python
dose_counts = df.groupby(['TREATMENT_ARM', 'ALPELISIB_DOSE']).size().unstack(fill_value=0)
# dose_counts.columns 是字符串类型: ['250', '300', '350']
```

**测试代码**：
```python
assert dose_counts.loc["A", 250] == 3  # 使用整数 250 索引
```

**根本原因**：
- `ALPELISIB_DOSE` 列是字符串类型
- `unstack()` 后列名保持字符串类型
- 但测试用整数索引，导致 KeyError

**解决方案**：
```python
# 方案1: 转换列名为整数
dose_counts.columns = dose_counts.columns.astype(int)

# 方案2: 在错误反馈中提供实际列名
# "KeyError: 250. Available columns: ['250', '300', '350'] (dtype: object)"
```

**是否环境问题**: ❌ 否
